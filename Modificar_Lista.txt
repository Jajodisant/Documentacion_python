*Cambiar el valor del elemento*

>>> Para cambiar el valor de un elemento específico, Consulte el número índice.

Ejemplo:
Cambiar el segundo elemento:
  thislist = ["apple", "banana", "cherry"]
  thislist[1] = "blackcurrant"
  print(thislist)

>>> Cambiar un rango de valores de artículo

Para cambiar el valor de los elementos dentro de un rango específico, defina una lista con los nuevos valores 
y consulte el rango de números de índice en el que desea insertar los nuevos valores.

Ejemplo
Cambie los valores "banana" y "cherry" con los valores "grosella negra" y "sandía".
  thislist = ["apple", "banana", "cherry", "orange", "kiwi", "mango"]
  thislist[1:3] = ["blackcurrant", "watermelon"]
  print(thislist)

>>> Insertar elementos

Para insertar un nuevo elemento de lista, sin sustituir ninguno de los valores existentes, podemos utilizar el insert()Método.
El insert()El método inserta un elemento en el índice especificado.

Ejemplo
Inserte "sandía" como tercer elemento.
  thislist = ["apple", "banana", "cherry"]
  thislist.insert(2, "watermelon")
  print(thislist)

>>> Añadir Lista de Artículos

Para añadir un elemento al final de la lista, utilice el append()Método.

Ejemplo
Usando el append()Método para añadir un elemento.
  thislist = ["apple", "banana", "cherry"]
  thislist.append("orange")
  print(thislist)
